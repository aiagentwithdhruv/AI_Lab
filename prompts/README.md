## How to Use Any Prompt

1. Open the prompt folder and read the `prompt.md` file.
2. Copy the full prompt into Claude Code (or any Claude interface with agent teams support).
3. The agents will self-organize based on the communication flow defined in the prompt.
4. Collect the deliverables listed in each prompt's final deliverables table.

Each prompt folder contains its own `README.md` with a detailed breakdown of agents, phases, and expected output.
